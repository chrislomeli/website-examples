Thank you for your purchase! Below is how to get started:

GET STARTED

1: Open the `simple-mdx-blog` in Visual Studio Code

2: Open the terminal in VSCode and run the command `nom install` to install all the modules from the `package.json` file

3: Run `nom run develop` to build a dev production site and visit the url at localhost.

4: Create a new Github Repo and follow the instructions to commit the code

5: Signup for Netlify and connect your Github account. Select your repo and let Netlify build your site!
